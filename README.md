#  Fabric Defect Detection using Deep Learning
A deep learning-based system for **automatic detection and classification of fabric defects** using Convolutional Neural Networks (CNNs).
The project aims to improve quality inspection in textile manufacturing by replacing manual inspection with an efficient, data-driven approach.

---

##  Overview
Fabric quality inspection is traditionally manual, time-consuming, and prone to inconsistencies.
This project leverages **computer vision and deep learning** to identify defects such as irregular textures, distortions, and structural inconsistencies in fabric images.
The model learns discriminative features and classifies images into **defective and non-defective categories**, along with specific defect types.

---

##  Features
*  Automated defect detection in fabric images
*  CNN-based feature learning (texture & pattern recognition)
*  Multi-class defect classification
*  Performance evaluation using accuracy & confusion matrix
*  Scalable for industrial textile inspection

## Model Details
* **Architecture:** Convolutional Neural Network (CNN)
* **Input:** Fabric images
* **Output:** Defect class / Normal fabric
* **Learning Focus:**
  * Texture variations
  * Pattern irregularities
  * Structural inconsistencies
